# NIS

NIS is a networked musical and performative work for mobile devices and real-time sound spatialization. The project explores listening, gesture, movement, and algorithmic mediation through smartphones acting as extensions of bodily attention and spatial perception.

Performers’ movements are mediated in real time through motion sensors and compass data integrated into mobile devices, generating an algorithmic sound spatialization system. The work investigates how technological mediation influences movement, performative decisions, and shared listening.

Developed within ongoing postdoctoral research supervised by Prof. Dr. Cássia Carrascoza.
